# AI Thread Exporter

## English

### Description
AI Thread Exporter is a Google Chrome extension designed to solve the limitation of context windows in AI models. While an AI may lose track of the beginning of a long conversation, the data remains available in the browser's DOM. This tool allows users to export entire chat histories from major AI platforms into structured Markdown (.md) files for local archiving and documentation.

### Features
* Universal Compatibility: Supports Google Gemini, OpenAI ChatGPT, and Anthropic Claude.
* Structured Formatting: Automatically converts chat elements into Markdown headers, lists, and code blocks.
* Metadata Integration: Each export includes YAML frontmatter with title, date, and source URL.
* Local Processing: All data extraction is performed locally within the browser to ensure user privacy.
* Character Encoding: Full support for international characters via UTF-8 encoding.

### Installation
1. Clone or download this repository to your local machine.
2. Open Google Chrome and navigate to `chrome://extensions/`.
3. Enable Developer Mode using the toggle in the upper right corner.
4. Click Load unpacked and select the directory containing the extension files.

### Usage
1. Navigate to a chat on Gemini, ChatGPT, or Claude.
2. Scroll through the history to ensure all messages are loaded in the browser.
3. Click the AI Thread Exporter icon in the extensions menu.
4. Select Export Thread to generate and download the Markdown file.

## Svenska

### Beskrivning
AI Thread Exporter är ett tillägg för Google Chrome skapat för att hantera begränsningar i AI-modellers kontextfönster. Trots att en AI kan tappa tråden i långa konversationer finns informationen kvar i webbläsarens kod. Detta verktyg gör det möjligt för användare att exportera hela chatthistoriker från ledande AI-plattformar till strukturerade Markdown-filer (.md) för lokal arkivering och dokumentation.

### Funktioner
* Universell kompatibilitet: Stöd för Google Gemini, OpenAI ChatGPT och Anthropic Claude.
* Strukturerad formatering: Konverterar automatiskt chattelement till Markdown-rubriker, listor och kodblock.
* Metadata: Varje export inkluderar YAML-frontmatter med titel, datum och käll-URL.
* Lokal bearbetning: All dataextrahering sker lokalt i webbläsaren för att säkerställa användarens integritet.
* Teckenkodning: Fullt stöd för svenska tecken via UTF-8-kodning.

### Installation
1. Klona eller ladda ner detta arkiv till din dator.
2. Öppna Google Chrome och gå till `chrome://extensions/`.
3. Aktivera Utvecklarläge via reglaget i det övre högra hörnet.
4. Klicka på Ladda obekräftat och välj mappen som innehåller tilläggets filer.

### Användning
1. Gå till en chatt på Gemini, ChatGPT eller Claude.
2. Scrolla igenom historiken för att säkerställa att alla meddelanden har laddats in i webbläsaren.
3. Klicka på ikonen för AI Thread Exporter i menyn för tillägg.
4. Välj Exportera tråd för att generera och ladda ner Markdown-filen.
